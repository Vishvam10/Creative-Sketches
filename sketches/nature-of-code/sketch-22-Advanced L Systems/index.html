Rules

  • Click on Add / Delete to create / delete a new rule
  • Rules are of the form : A -> B ( case insensitive )
  • Use only the symbols present in the language
  • Use only the arrow symbol (->) to separate the LHS and RHS
  • Once done, click on Next to update the generation

Examples

  • Example 1

    Rule 1 :
    F -> FF+[+F-F-F]-[-F+F+F]

  • Example 2

    Rule 1 :
    F -> F+[F-F]+F
    Rule 2 :
    FF -> F-F
    Rule 3 :
    F+ -> FF+FF

  • Example 3

    Rule 1 :
    F -> FF+[F-F]+[F+F]
    Rule 2 :
    FF -> F-F

Language :

'F', '+', '-', '[', ']'

Legend :

F -> Translate to that point

+ -> Rotate Clockwise

- -> Rotate Anti-Clockwise

[ -> Persist state

] -> Discard state
